Un carnet de recherche personnel documentant l'apprentissage systématique des mathématiques fondamentales, avec un focus sur l'algèbre linéaire, l'analyse et leurs applications en cryptologie.

## 📚 Contenu

Ce cahier couvre les domaines suivants :

- **Suites numériques** : convergence, limites, suites arithmétiques et géométriques
- **Fonctions** : limites, continuité, théorème des valeurs intermédiaires
- **Espaces vectoriels ℝⁿ** : structure, sous-espaces, combinaisons linéaires
- **Applications linéaires** : noyau, image, rang, théorèmes fondamentaux
- **Systèmes linéaires** : méthode de Gauss, résolution, interprétation géométrique

## 🎯 Objectif

Construire une référence personnelle structurée pour :
- Maîtriser les fondations mathématiques nécessaires en cryptologie
- Documenter exercices et solutions de manière systématique
- Créer un support de révision visuel et hiérarchisé
